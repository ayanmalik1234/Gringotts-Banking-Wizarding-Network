# Gringotts Banking Wizarding Network

A dependency-free simulated fintech and trading dashboard with:

- multi-currency conversion for Galleons, Sickles, Knuts, USD, EUR, and GBP
- a candlestick-style canvas market chart
- a vault allocation modeler for fictional commodities
- a downloadable transaction invoice ledger
- a live vault security log feed

Open `index.html` in a browser to run it locally.
